# Password Strength Checker 🔐

A simple Python project to check password strength based on:

- Minimum length
- Uppercase & lowercase letters
- Numbers
- Special characters

## How to Run
1. Save the file as password_strength_checker.py
2. Run the script:
```bash
python password_strength_checker.py


## 📥 Sample Output
